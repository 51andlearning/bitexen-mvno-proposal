# Scope

## Pages Included

1. **Home / Hero** — Strategic overview, headline value proposition
2. **The Opportunity** — SA MVNO market context, why now, why Bitexen
3. **Fintech Arsenal** — Bitexen's full crypto/fintech portfolio mapped to MVNO value
4. **Product Tiers** — Bitexen Connect / Bitexen Plus / Bitexen Black tier breakdown
5. **Crypto Rewards & Trading Strategy** — Exchange integration, rewards programme, trading tools differentiator
6. **Property Strategy Map** — Each Bitexen product mapped to telecom product and segment
7. **Revenue Model** — Layered revenue stack, ARPU projections, customer value flywheel
8. **Next Steps** — Implementation workstreams, timeline, call to action

## Sections Included

- Navigation header with anchor links
- Hero section with key statistics
- Content cards for each Bitexen division
- Interactive tier comparison (tabs or cards)
- Trading tools and crypto rewards showcase
- Data tables for user reach and ARPU
- Revenue stack visualisation
- Contact / next steps CTA
- Footer

## What Is Excluded

- User authentication / login
- Payment processing
- CMS or admin panel
- Blog or news feed (live content)
- Analytics dashboard
- API integrations with Bitexen systems
- Multi-language support (English only initially; Turkish localisation post-launch)
