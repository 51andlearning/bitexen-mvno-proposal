# Project Overview

## What This Is

A standalone proposal website for the **Bitexen Fintech MVNO** — a fintech-led mobile virtual network operator leveraging Bitexen's cryptocurrency exchange platform, 2M+ active user base, and digital wallet ecosystem to create a differentiated telecoms offering in South Africa.

## Purpose

This site serves as:
- A digital proposal document for Bitexen decision-makers
- A visual representation of the MVNO strategy, product tiers, and fintech integration model
- A leave-behind asset for board-level and executive stakeholders
- A demonstration of the digital capability that would underpin the MVNO's customer experience

## Target Audience

- Bitexen executive leadership and board
- Bitexen divisional heads (Exchange, Wallet, Research, Technology)
- Potential MNO partners (Cell C, MTN)
- Digital Mobile / MVNE partners
- Investors and strategic advisors

## Technology Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript

- **Components:** shadcn UI
- **Deployment:** Vercel
